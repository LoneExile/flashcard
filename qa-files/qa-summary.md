## Key Features

### Multiple Ways to Ask Questions
Each topic includes various ways to ask the same question, for example:
- "nǐ jīn tiān zuò shén me?" vs "jīn tiān nǐ zuò shén me?"
- "nǐ zǎo shàng jǐ diǎn qǐ chuáng?" vs "nǐ shàng wǔ jǐ diǎn qǐ chuáng?"

### Multiple Ways to Answer
Each section provides various answer patterns using different vocabulary, for example:
- Time: "zǎo shàng" vs "shàng wǔ" (both mean morning)
- Location: Different cities (běi jīng, shàng hǎi, guǎng zhōu, xiāng gǎng)
- Positive and negative forms

### Vocabulary Sources
All vocabulary used in these Q&A files comes exclusively from the lesson files (lesson-01.txt through lesson-32.txt) in the data directory, including:

## Usage Instructions

These Q&A files can be used for:
1. **Self-study**: Practice asking and answering questions in Mandarin
2. **Conversation practice**: Use with a language partner or tutor
3. **Flashcard creation**: Convert Q&A pairs into flashcards
4. **Assessment**: Test comprehension and speaking abilities
5. **Role-play scenarios**: Simulate real-life conversations
